The file "scriptForRunning" contains the format for running the snake on the set of images provided.
Once you save the files in a directory, the paths have to be updated accordingly.
for_file_name is for the file names of the pictures, in general.
pathForDir is the variable which will take care of sellecting all the files which end with *.jpg
outputPath is where I am saving the outputs.

mat_file are the template files.
template for first set of liptracking: testInputs.mat
template for second set of liptracking: templateForSecondSet.mat
template for third set of liptracking: templateForThirdSet.mat
